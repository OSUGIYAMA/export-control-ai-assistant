## 🎯 基本的な使い方

### 1. 契約書を分析する

1. ブラウザで `http://localhost:8501` を開く
2. 「📄 契約書分析」タブを選択
3. **方法A: PDFアップロード**
   - 輸出契約書PDFをアップロード
   - 「🔍 分析開始」をクリック
   
4. **方法B: 手動入力**
   - 「または、契約情報を手動で入力」を展開
   - 必要情報を入力（品目名、仕向地、需要者等）
   - 「🔍 分析開始」をクリック

5. 分析結果を確認：
   - ✅ リスクレベル（高/中/低）
   - ✅ 抽出された契約情報
   - ✅ AI分析詳細
   - ✅ 推奨アクション

6. 結果をダウンロード：
   - 「📥 分析結果をダウンロード」
   - 「📥 詳細レポートをダウンロード」

### 2. 輸出管理について質問する

1. 「💬 チャット相談」タブを選択
2. 質問を入力（例：「グループA国とは？」）
3. 「送信」をクリック
4. AIが回答を生成

### 3. 規制データを確認する

1. 「📊 データ管理」タブを選択
2. 表示するデータを選択：
   - ECCN番号リスト
   - カントリーグループ
   - エンティティリスト（サンプル）
3. データを確認・検索

## 🔧 よくある質問

### Q1: アプリが起動しない
**A**: 以下を確認してください：
```bash
# 依存関係を再インストール
pip install -r requirements.txt

# アプリを起動
streamlit run app.py
```

### Q2: API keyエラーが出る
**A**: `.env`ファイルを確認してください：
```
OPENAI_API_KEY=sk-proj-...（あなたのAPIキー）
```

### Q3: PDFがうまく読み込めない
**A**: 以下を試してください：
- OCR処理済みのPDFを使用
- 手動入力機能を利用
- テキスト形式でコピー＆ペースト

### Q4: 分析結果が不正確
**A**: 
- 手動入力で正確な情報を提供
- チャット機能で詳細を確認
- 専門家に相談
