# 🎨 デザインノート - Clean White Flat UI

## デザインコンセプト

**テーマ**: クリーン・ホワイト・フラット
**カラーパレット**: シンプルな青系アクセント（#3182ce）

---

## 主要カラー

### ベースカラー
- **背景色**: `#ffffff` (Pure White)
- **テキスト**: `#1a202c` (Dark Gray)
- **セカンダリテキスト**: `#4a5568` (Medium Gray)

### アクセントカラー
- **プライマリ**: `#3182ce` (Clean Blue)
- **プライマリダーク**: `#2c5282` (Hover State)
- **成功**: `#10b981` (Emerald Green)
- **警告**: `#f39c12` (Orange)
- **エラー**: `#ef4444` (Red)
- **情報**: `#3182ce` (Blue)

### ニュートラルカラー
- **ボーダー**: `#e2e8f0` (Light Gray)
- **背景グレー**: `#f7fafc` (Very Light Gray)
- **ホバーグレー**: `#cbd5e0` (Medium Light Gray)

---

## デザイン原則

### 1. フラットデザイン
- グラデーション不使用
- シンプルな単色背景
- 控えめなシャドウ
- クリーンなボーダー

### 2. 視認性
- 高コントラスト（白背景に黒文字）
- 読みやすいフォントサイズ
- 十分な行間とスペーシング

### 3. ミニマリズム
- 装飾を最小限に
- 必要な要素のみ表示
- シンプルなレイアウト

## UIコンポーネント

### ヘッダー
```css
font-size: 2.5rem
font-weight: 700
color: #1a202c
border-bottom: 3px solid #3182ce
```

### ボタン
```css
背景: #3182ce (青)
ホバー: #2c5282 (濃い青)
角丸: 6px
パディング: 0.625rem 1.5rem
```

### アラートボックス
```css
フラットな背景色
左ボーダー: 4px solid
角丸: 4px
控えめな色使い
```

### 入力フィールド
```css
ボーダー: 1px solid #cbd5e0
フォーカス: #3182ce
角丸: 6px
シンプルなデザイン
```

### タブ
```css
背景: 透明
選択時: #3182ce (青)
ボーダーなし
シンプルな切り替え
```

### サイドバー
```css
背景: #f7fafc (薄いグレー)
ボーダー: 1px solid #e2e8f0
```

## カスタマイズ方法

### プライマリカラー変更
`app.py`の以下を変更：
```css
#3182ce  → 任意の色
```

### ボーダーの太さ
```css
border: 1px solid #e2e8f0;
       ↑ この値を変更
```

### 角丸の調整
```css
border-radius: 6px;
              ↑ この値を変更
```
